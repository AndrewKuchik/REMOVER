# ✂️ REMOVER — удаление фона в браузере

Бесплатный удалитель фона, который работает **прямо в браузере**: без сервера, без оплаты,
без регистрации. Картинки никуда не отправляются — всё считается на твоём устройстве.

- 🎯 Убирает фон у любых картинок, **сохраняя полное разрешение**.
- 🧹 Автоматически убирает **цветную каёмку** (деконтаминация края) — то, чего нет во встроенных тулах.
- 🎚️ Ползунки: обрезка каёмки, мягкость края, очистка цвета.
- 🖼️ Предпросмотр на любом фоне (белый/чёрный/цвет) и ползунок «до/после».
- ⚡💎 Два режима: **Быстро** (@imgly) и **Качество** (BiRefNet на WebGPU) для сложных кадров.

## Как запустить у себя на компьютере

Нужен установленный [Node.js](https://nodejs.org) (LTS).

```bash
npm install     # один раз — скачать зависимости
npm run dev     # запустить; открой ссылку, которую покажет терминал (обычно http://localhost:5173)
```

Готово: перетащи картинку в окно → скачай PNG без фона.

> При первом вырезании браузер один раз докачивает модель (быстрый режим ~40–80 МБ,
> режим «Качество» — модель BiRefNet). Потом она берётся из кэша и работает быстро.

## Как это устроено (коротко)

1. **Движок-мозг** считает маску (где объект, где фон):
   - быстрый режим — `@imgly/background-removal`;
   - режим «Качество» — `BiRefNet` через `@huggingface/transformers` (WebGPU, с откатом на WASM).
2. **Наш слой обработки края** (`src/refine.js`) накладывает маску на оригинал в полном
   разрешении и чистит край: эрозия + «затекание» цвета объекта наружу + мягкость.
